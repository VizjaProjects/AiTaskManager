# Ordovita Marketing — UI kit

A recreation of the public `ordovita.pl` landing page. Recreated from
`client/app/index.tsx`.

## Open

`index.html` — renders the full landing page.

## Screens

- `LandingScreen.jsx` — header, serif hero with one accent word, desktop download buttons, app preview, three feature cards, footer.

## Notes

- Unlike in-app surfaces (which are flat), the marketing page uses a soft real shadow (`--shadow-landing`) so white cards lift off the cream paper.
- One accent only; the hero is Playfair Display, body is Inter.
- Cosmetic recreation — buttons are non-functional.
